# Deep Sea Hunter: Bullet Hell 🌊🏹

**Deep Sea Hunter** — это динамичный аркадный шутер в жанре **Bullet Hell**, разработанный на Python с использованием библиотеки [Pygame](https://www.pygame.org/). Вы играете за отважного аквалангиста, который погрузился в самые опасные глубины океана, чтобы сразиться с ордами агрессивных рыб и морских чудовищ.

## 📖 Сюжет
Глубоководные жемчужины охраняются древними обитателями океана. Ваш аквалангист вооружен лишь верным гарпуном и ограниченным запасом кислорода. Сможете ли вы выжить в «шторме» из вражеских снарядов и собрать сокровища бездны?

## 🎮 Игровой процесс
* **Уворачивайтесь:** Сотни икринок-пуль, электрических разрядов и ядовитых пузырей заполняют экран.
* **Охотьтесь:** Используйте гарпун, чтобы пробивать путь сквозь стаи врагов.
* **Улучшайтесь:** Собирайте бонусы (Power-ups), чтобы стрелять быстрее, мощнее или восполнять запас кислорода.
* **Боссы:** Каждые 5 уровней вас ждет встреча с легендарным морским гигантом.

## 🕹 Управление
* **WASD / Стрелки** — Перемещение аквалангиста.
* **Пробел (Space) / ЛКМ** — Выстрел из гарпуна.
* **Left Shift** — Фокусировка (замедление движения для точного маневрирования между пулями).
* **P** — Пауза.
* **Esc** — Выход в главное меню.

## 🛠 Установка и запуск

1.  **Клонируйте репозиторий:**
    ```bash
    git clone [https://github.com/yourusername/deep-sea-hunter.git](https://github.com/yourusername/deep-sea-hunter.git)
    cd deep-sea-hunter
    ```

2.  **Установите зависимости:**
    Убедитесь, что у вас установлен Python 3.8 или выше. Установите библиотеку Pygame:
    ```bash
    pip install pygame
    ```

3.  **Запустите игру:**
    ```bash
    python main.py
    ```
