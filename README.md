## ⚓️ Описание проекта

Игра про выживание дайвера в агрессивной глубоководной среде. Главная задача — продержаться 5 минут вокруг врагов-торпед, которые перемещаются случайным образом.

### Реализованные фичи:

- **Скриптовые атаки:** Враги не просто стреляют в игрока, а выдают сложные паттерны (спирали, кольца, веерные залпы).
- **Focus Mode:** При зажатии `Shift` скорость дайвера падает, позволяя точнее маневрировать в узких просветах между пулями.
- **Боссы:** Каждые 5 уровней появляется крупный противник с несколькими фазами поведения.
- **Оптимизация:** Автоматическое удаление объектов за границами экрана для предотвращения утечек памяти.
